# Contributing to AgentFlow

## Consumer feedback

If you build something on top of `@ageflow/*` and hit an API gap, please:

1. Open an issue with the `consumer-feedback` label.
2. Describe the workaround you used (so we can mark it as accepted-as-design or fix it).
3. We track these in `KNOWN_CONSUMER_GAPS.md` and aim to close or formally accept within 2 weeks.

See [KNOWN_CONSUMER_GAPS.md](./KNOWN_CONSUMER_GAPS.md) for current status.
